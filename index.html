Image Type: Visual representation of uploaded image.

Music Type: Select the musical style for the accompaniment. For example, you can choose from Pop, Rock, EDM, Jazz, Waltz, Blues, etc. The chosen style will automatically determine the default time signature and corresponding pattern options.

Time Signature: Choose the time signature for the piece, such as 4/4, 3/4, 2/4, 6/8, 5/4, 7/8, 9/8, or 12/8. This defines the number of beats per measure and sets the overall rhythmic feel of the accompaniment.

Num of Measures: Specify the total number of measures for the generated accompaniment. This parameter controls the length of the accompaniment (e.g., 4 measures, 8 measures, etc.).

Method: Select the method for generating the accompaniment. If you choose "pattern", you will be able to manually select a specific accompaniment pattern from the list below; otherwise, the accompaniment will be generated automatically.

Pattern Name: When the Method is set to "pattern", use this dropdown to choose a specific accompaniment pattern (e.g., pop_4_4, rock_4_4, edm_4_4, jazz_4_4, etc.). This field will be enabled only if the Method is "pattern"; otherwise, it remains disabled.

Left Instrument: Select the instrument for the left-hand part (or lower voice), typically used for bass or harmonic accompaniment, such as a bass guitar or left-hand piano.

Right Instrument: Select the instrument for the right-hand part (or higher voice), typically representing the lead melody or primary accompaniment, such as a lead guitar, piano, or vocal backing.
